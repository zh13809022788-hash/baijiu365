# Roadmap

## Sprint 1: MVP Foundation

Goal: create a credible English static site that can be previewed locally and later deployed to Cloudflare Pages.

Deliverables:

- Astro project setup
- Homepage
- Base layout and navigation
- Article/guide layout
- First 6-8 core pages
- Glossary entry structure
- About/editorial standards page
- Sitemap and robots
- Build verification

Success criteria:

- `npm run build` passes
- Pages are readable on desktop and mobile
- Site does not feel like a placeholder landing page
- Content has useful judgment, not generic AI filler
- Development decisions are recorded in `docs/DEVELOPMENT_LOG.md`

## Sprint 2: Content Credibility

Goal: make the site feel useful and differentiated.

Deliverables:

- Expand to 12 core guides
- Add flavor framework
- Add beginner bottle selection framework
- Add fake/premium baijiu risk guide
- Add FAQ blocks to core pages
- Add JSON-LD where appropriate
- Add source notes / editorial disclosure where useful

## Sprint 3: Publishing and Indexing

Goal: deploy and make the site indexable.

Deliverables:

- Cloudflare Pages deployment
- Domain setup
- Search Console setup
- Sitemap submission
- Basic analytics
- First technical SEO pass
- Initial internal linking pass

## Sprint 4: WORKBUDDY Input Loop

Goal: turn domestic market knowledge into differentiated English content.

Deliverables:

- Domestic topic watchlist
- Baijiu pricing/channel notes
- Real buyer questions
- Gift/banquet/cocktail use cases
- Initial brand/product data format
